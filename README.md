# Scala-99
My codes for [scala99](http://aperiodic.net/phil/scala/s-99/)
naively written with somewhat functional style

### Notes for myself

* :\ foldRight
* /: foldLeft
* ++ concat anything
* +: prepend element
* :+ postpend element
* <: derive from
* `Tuple2._1 Tuple2._2`
* Right/Left Either
* `span` seperates a list by using predicate
* `zip` and `zipWithIndex`
* 对于trivial 的匿名函数可以 _
* :::是取出List元素拼接，::是直接拼接List
